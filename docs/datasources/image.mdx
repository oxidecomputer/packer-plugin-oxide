## Examples

Fetch a project image.

```hcl
data "oxide-image" "example" {
  name    = "ubuntu"
  project = "oxide"
}
```

Fetch a silo image.

```hcl
data "oxide-image" "example" {
  name = "ubuntu"
}
```
